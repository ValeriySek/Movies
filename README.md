# Movies
Приложение отображающее фильмы из базы данных. Приложение построено на Single-Activity с применением принципов Material Design

Использованные библиотеки и архитектура
--------------
  * Data Binding
  * LiveData
  * Navigation
  * Room
  * ViewModel
  * Picasso
  * Retrofit
  * RxJava2
  * Gson
